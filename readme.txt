English:

Get the WordReference's translation for the selected text.

Compatible with Firefox and Thunderbird.

WordRefence Translator allows to access to the translation of the selected word through 4 different ways:
* If you select it with double click, it will appear a link over it.
* And even if you don't select it with double click:
** Pressing the key "D"
** With the option "Translate" in the context menu
** With the option "Translate" in the "Tools" menu
        


Castellano:

Obtén la traducción de WordReference para el texto seleccionado.

Compatible con Firefox y Thunderbird.

WordRefence Translator permite acceder a la traducción de la palabra seleccionada de 4 formas diferentes:
* Si la seleccionas con doble clic, aparecerá un enlace sobre ella.
* Y aunque no la selecciones con doble clic:
** Pulsando la tecla "D"
** Con la opción "Traduce" en el menú contextual
** Con la opción "Traduce" en el menú "Herramientas"
